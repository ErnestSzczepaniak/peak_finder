# Kompilacja i uruchamianie

`make`

`./peak_finder.elf size seed algorithm`

* `size` rozmiar generowanej mapy
* `seed` ziarno do generowania mapy (jak w minecraft), 
* `algorithm` algorytm - `climb` lub `greedy`

# Przyklad wywolania

`./peak_finder.elf 50 100 climb`
